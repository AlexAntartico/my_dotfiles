# current/

Active CachyOS setup. Files in this directory are symlinked into `$HOME` by `../install.zsh`.

The structure here mirrors `$HOME`:

- `.zshrc` → `~/.zshrc`
- `.tmux.conf` → `~/.tmux.conf`
- `.config/alacritty/alacritty.toml` → `~/.config/alacritty/alacritty.toml`
- `.config/starship.toml` → `~/.config/starship.toml`
- `.config/zsh/cachyos-config.zsh` → `~/.config/zsh/cachyos-config.zsh`

See the top-level [README](../README.md) for installation instructions and the rationale behind the two-layer zsh config.
